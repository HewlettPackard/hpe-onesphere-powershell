	Powershell bindings for the OneSphere REST API


	WHAT'S NEW




	Powershell bindings for the OneSphere REST API



	Import Module with: import-module hpeonesphere List available commands with: get-command -module hpeonesphere



	Check hpeonesphere-tests.ps1 for the rest.



	Use with .\samples\hpeos-tests.ps1 -portal $YourOneSpherePortal -username $YourOneSphereUsername -password $YourOneSpherePassword -short
